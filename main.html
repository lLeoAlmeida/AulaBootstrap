Introdução ao framework Bootstrap

Hoje irei apresentar os conceitos básicos do Bootstrap, um dos frameworks CSS mais populares e amplamente utilizados no desenvolvimento front-end.

O que é um framework?

Antes de falar um pouco sobre o Bootstrap, primeiramente precisamos entender o que é um framework.

Imagina que você está tentando contruir um carro utilizando lego (Bloquinhos de construção). Contudo, Fazer tudo do zero pode ser muito difícil e demorado. Um framework é como se fosse alguns bloquinhos pré-montados que ajudam você a construir o carro mais rapidamente e com mais eficiência. É como se alguém já tivesse feito uma base, e você não precisaria começar a montar teu carro do zero, você pode, por exemplo começar com uma estrutura do motor já pronta e só costumizar as outras partes como quiser.
É por isso que os desenvolvedores usam frameworks para construir aplicativos e sites de maneira mais fácil e rápida.

Bootstrap

O Bootstrap é um framework CSS de código aberto que foi criado para facilitar o desenvolvimento de sites responsivos e compatíveis com diferentes dispositivos, como desktops, laptops, tablets e smartphones. Ele fornece uma série de componentes pré-construídos, como botões, formulários, menus de navegação, grades e muito mais, que podem ser facilmente integrados em um site existente. O Bootstrap foi desenvolvido pela equipe do Twitter e é amplamente utilizado por desenvolvedores em todo o mundo. Ele usa o conceito de grid (grade) para organizar o conteúdo do site em colunas e linhas, permitindo que os desenvolvedores criem layouts complexos com facilidade.
